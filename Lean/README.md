# CS5232: Accompanying Lean files

## Prerequisites

We will be using VSCode to edit Lean files. Follow [these instructions](https://lean-lang.org/install/) to get the necessary toolset.

## Building the Code

Run from the command line

```
lake update
lake build
```

## Project Structure

* `LoVe` -- demo code and exercises from the book [The Hitchhiker's Guide to Logical Verification](https://github.com/lean-forward/logical_verification_2025/) by Baanen et al.
* `PraVDA` -- examples of Practical Verified Data structures and Algorithms using [Loom/Velvet](https://github.com/verse-lab/loom) and [Veil](https://github.com/verse-lab/veil) frameworks 
* `Project` -- a placeholder folder where you should put the implementation of your course project.
